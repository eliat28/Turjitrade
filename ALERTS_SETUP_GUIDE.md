# 📧 מדריך הגדרת התראות TurjiTrade

## סקירה כללית

TurjiTrade תומך בשליחת התראות אוטומטיות דרך **מייל** ו-**טלגרם** כאשר מניה מגיעה לטווח המחיר המבוקש.

---

## 🔧 הגדרת התראות מייל (EmailJS)

EmailJS הוא שירות חינמי (עד 200 מיילים/חודש) שמאפשר לשלוח מיילים ישירות מהדפדפן.

### שלב 1: הרשמה ל-EmailJS

1. היכנס ל-[EmailJS.com](https://www.emailjs.com)
2. לחץ על "Sign Up" והירשם (חינם)
3. אשר את המייל שלך

### שלב 2: יצירת Email Service

1. לחץ על "Email Services" בתפריט הצד
2. לחץ על "Add New Service"
3. בחר את ספק המייל שלך (Gmail, Outlook, Yahoo וכו')
4. התחבר לחשבון המייל שלך
5. **העתק את ה-Service ID** (לדוגמה: `service_abc123`)

### שלב 3: יצירת Email Template

1. לחץ על "Email Templates" בתפריט הצד
2. לחץ על "Create New Template"
3. עצב את התבנית שלך עם המשתנים הבאים:

```html
נושא: התראת קנייה - {{symbol}} | TurjiTrade

גוף ההודעה:
🎯 זמן מעולה покупа!

המניה {{symbol}} נמצאת בטווח המחיר שהגדרת:

📊 מחיר נוכחי: ${{current_price}}
📉 טווח: ${{price_min}} - ${{price_max}}

זה הזמן לשקול קנייה!

🤖 TurjiTrade - מסחר חכם עם AI
```

4. **העתק את ה-Template ID** (לדוגמה: `template_xyz789`)

### שלב 4: קבלת Public Key

1. לחץ על "Account" בתפריט הצד
2. עבור ל-"API Keys"
3. **העתק את ה-Public Key** (לדוגמה: `abcXYZ123`)

### שלב 5: הזנה ב-TurjiTrade

1. פתח את TurjiTrade
2. עבור ל-**טאב "התראות"**
3. לחץ על כפתור **⚙️ הגדרות** בפינה הימנית העליונה
4. בחר טאב **"Email (EmailJS)"**
5. הזן את:
   - **Service ID**
   - **Template ID**
   - **Public Key**
6. לחץ **"שמור הגדרות"**

## 📱 הגדרת התראות טלגרם (Telegram Bot)

טלגרם מאפשר יצירת בוטים בחינם ללא הגבלה. המערכת מציעה חיבור פשוט ונוח!

### אופציה 1: חיבור פשוט (מומלץ!) ⭐

**למשתמשים - ממש פשוט!**

1. פתח את TurjiTrade
2. עבור ל-**טאב "התראות"**
3. לחץ על **"התחבר דרך Telegram"**
4. בטלגרם, שלח `/start` לבוט
5. **העתק את ה-Chat ID** שהבוט שולח לך
6. **הדבק** את המספר באפליקציה
7. לחץ **"שמור Chat ID"**

**זהו!** עכשיו תקבל התראות! 🎉

---

### אופציה 2: הגדרה ידנית

אם אתה רוצה להשתמש בבוט משלך:

### שלב 1: יצירת Telegram Bot

1. פתח את Telegram
2. חפש את **@BotFather**
3. שלח לו `/newbot`
4. בחר שם לבוט (לדוגמה: "TurjiTrade Alert Bot")
5. בחר username לבוט (חייב להסתיים ב-`bot`, לדוגמה: `turjitrade_alert_bot`)
6. **BotFather יחזיר לך Bot Token** - **העתק אותו!**
   - נראה כך: `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`

### שלב 2: קבלת Chat ID

1. שלח הודעה כלשהי לבוט שלך (הודעה ראשונה)
2. חפש בטלגרם את **@userinfobot**
3. שלח לו הודעה כלשהי
4. הוא יחזיר לך את ה-**Chat ID** שלך (לדוגמה: `123456789`)
5. **העתק את ה-Chat ID**

### שלב 3: הזנה ב-TurjiTrade

1. פתח את TurjiTrade
2. עבור ל-**טאב "התראות"**
3. לחץ על כפתור **⚙️ הגדרות** בפינה הימנית העליונה
4. בחר טאב **"Telegram Bot"**
5. הזן את ה-**Bot Token**
6. לחץ **"שמור הגדרות"**

### שלב 4: הזנת Chat ID לכל התראה

בעת הוספת מניה לרשימת המעקב עם התראת קנייה:

1. בחר **"טלגרם"** כערוץ התראה
2. הזן את ה-**Chat ID** שקיבלת מ-@userinfobot
3. שמור את המניה

- `TELEGRAM_USER_GUIDE.md` - מדריך למשתמשים

## 📋 שימוש בהתראות

### הוספת התראת קנייה

1. עבור ל-**טאב "רשימת מעקב"**
2. לחץ על **➕** להוספת מניה
3. הזן את סימבול המניה
4. סמן **"התראת קנייה"**
5. הזן:
   - **מחיר מינימום** - המחיר הנמוך ביותר לקנייה
   - **מחיר מקסימום** - המחיר הגבוה ביותר לקנייה
6. בחר ערוצי התראה:
   - ☑️ **מייל**
   - ☑️ **טלגרם**
7. לחץ **"הוסף למעקב"**

### איך זה עובד?

- המערכת בודקת מחירים כל **30 שניות**
- כאשר המחיר נכנס לטווח שהגדרת:
  - **נוטיפיקיישן ירוק** יופיע בראש המסך
  - **מייל** יישלח לכתובת שהגדרת
  - **הודעת טלגרם** תישלח לחשבון שלך
- ההתראה תישלח **רק פעם אחת** עבור כל מניה
- לאיפוס ההתראה, מחק את המניה והוסף אותה מחדש

## ❓ שאלות נפוצות

### האם זה בטוח?

כן! כל המפתחות נשמרים רק **בדפדפן שלך** (localStorage) ולא נשלחים לשום שרת.

### האם זה חינמי?

כן! שני השירותים מציעים תוכניות חינמיות:
- **EmailJS**: 200 מיילים/חודש
- **Telegram**: ללא הגבלה

### מה אם ההתראה לא מגיעה?

1. בדוק שהגדרת נכון את ה-API keys
2. בדוק שהזנת נכון את המייל/Chat ID
3. בדוק את תיקיית הספאם במייל
4. בדוק שהבוט קיבל לפחות הודעה אחת ממך

### איך אני משנה את המייל/Chat ID?

1. עבור ל-**טאב "התראות"**
2. לחץ על האייקון ליד **מייל** או **טלגרם**
3. עדכן את הפרטים
