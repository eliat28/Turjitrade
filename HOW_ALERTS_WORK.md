# 🔔 איך מערכת ההתראות עובדת - TurjiTrade

## 🎯 סקירה כללית

TurjiTrade שולחת לך התראות **אוטומטיות** כשמניה מגיעה לטווח המחיר שהגדרת!  
ההתראות נשלחות דרך **מייל** ו/או **טלגרם** - לבחירתך.

---

## 📱 איך זה עובד? (פשוט!)

### שלב 1️⃣: הגדר את פרטי הקשר שלך (פעם אחת!)

#### **למייל:**
1. עבור ל-**טאב "התראות"** 📧
2. לחץ על **"הגדר כתובת מייל"**
3. הזן את המייל שלך
4. לחץ **"שמור"**

#### **לטלגרם:**
1. עבור ל-**טאב "התראות"** 📱
2. לחץ על **"התחבר דרך Telegram"**
3. לחץ על **"פתח את הבוט"**
4. שלח **`/start`** בטלגרם
5. **העתק** את ה-Chat ID שהבוט שולח
6. **הדבק** באפליקציה
7. לחץ **"שמור Chat ID"**

✅ **זהו!** עכשיו הפרטים שלך שמורים!

---

### שלב 2️⃣: הוסף מניה עם התראת קנייה

1. עבור ל-**טאב "רשימת מעקב"** 📊
2. לחץ על **➕** (כפתור הוספה)
3. הזן את **סימבול המניה** (לדוגמה: AAPL)
4. **סמן** את ✅ **"התראת קנייה"**
5. הזן את **טווח המחיר** שברצונך לקנות:
   - **מחיר מינימום** - המחיר הנמוך ביותר לקנייה (לדוגמה: 150)
   - **מחיר מקסימום** - המחיר הגבוה ביותר לקנייה (לדוגמה: 180)
6. **בחר ערוצי התראה:**
   - ☑️ **מייל** - קבל התראה למייל
   - ☑️ **טלגרם** - קבל התראה לטלגרם
7. לחץ **"הוסף למעקב"**

✅ **סיימת!** המניה נוספה!

---

### שלב 3️⃣: המתן להתראה (אוטומטי!)

**המערכת עושה את כל העבודה בשבילך:**

1. ✅ **בודקת מחירים כל 30 שניות** אוטומטית
2. ✅ **מזהה** כשמניה נכנסת לטווח המחיר
3. ✅ **שולחת התראות** למייל ו/או טלגרם
4. ✅ **מציגה נוטיפיקיישן ירוק** בראש המסך

## 💡 טיפים והמלצות

### ✅ **DO:**
- הגדר טווח מחירים **סביר** (לא צר מדי)
- השתמש **בשני הערוצים** (מייל + טלגרם) לגיבוי
- בדוק שהמייל/טלגרם שלך **פעילים**
- **בדוק התראה** לפני שאתה מסתמך עליה (כפתור "בדוק התראה עכשיו")

### ❌ **DON'T:**
- אל תגדיר טווח **צר מדי** (קשה להגיע)
- אל תגדיר טווח **רחב מדי** (התראה מיידית)
- אל תשכח לשמור את המייל/Chat ID
- אל תסתמך **רק** על התראות - תמיד בדוק בעצמך!

## 🔧 פתרון בעיות

### ❓ **ההתראה לא הגיעה?**

**בדוק:**
1. ✅ המייל/Chat ID שמורים ב-**טאב "התראות"**
2. ✅ המחיר **באמת** נכנס לטווח שהגדרת
3. ✅ לא כבר נשלחה התראה (בדוק `buyAlertTriggered`)
4. ✅ ההתראה לא בתיקיית **ספאם** (במייל)
5. ✅ הבוט של טלגרם **רץ** (למפתחים)

### ❓ **שגיאה בשליחת מייל?**

**בדוק:**
1. ✅ EmailJS מוגדר נכון ב-`apiConfig.ts`
2. ✅ יש לך פחות מ-**200 מיילים/חודש** (מגבלת EmailJS)
3. ✅ כתובת המייל **תקינה**

### ❓ **שגיאה בשליחת טלגרם?**

**בדוק:**
1. ✅ שלחת **`/start`** לבוט (@turji_trade_bot)
2. ✅ ה-Chat ID **נכון** (רק ספרות)
3. ✅ הבוט Python **רץ** (למפתחים: `python telegram_bot.py`)
4. ✅ ה-Bot Token נכון ב-`apiConfig.ts`
