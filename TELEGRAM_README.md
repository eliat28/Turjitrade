# 📚 TurjiTrade - מדריכי התראות טלגרם

## 🎯 סקירה כללית

TurjiTrade כולל מערכת התראות מתקדמת עם תמיכה ב-**מייל** (EmailJS) ו-**טלגרם** (Telegram Bot).  
המערכת עברה שדרוג לגרסה **פשוטה ונוחה** שבה המשתמש פשוט מעתיק ומדביק את ה-Chat ID!

## 🔧 מה השתנה?

### ✅ הגרסה החדשה (פשוטה!):
- ✅ המשתמש פשוט מעתיק/מדביק Chat ID
- ✅ אין צורך ב-Backend polling
- ✅ אין צורך ב-Webhook endpoints
- ✅ הבוט פשוט מציג את ה-Chat ID
- ✅ UI ברור ופשוט

### ❌ הגרסה הישנה (מורכבת):
- ❌ polling אוטומטי כל שנייה
- ❌ Supabase endpoints מורכבים
- ❌ תלות בשרת Backend
- ❌ timeout של 60 שניות
- ❌ מסובך למשתמשים
