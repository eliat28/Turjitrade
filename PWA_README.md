# 📱 TurjiTrade - PWA Quick Start

## התקנת האפליקציה למסך הבית

### 🤖 אנדרואיד (Chrome)
1. פתח את TurjiTrade בדפדפן
2. לחץ על כפתור "התקן עכשיו" בחלק התחתון של המסך
3. או: תפריט (⋮) → "הוסף למסך הבית"

### 🍎 iOS (Safari)
1. פתח את TurjiTrade ב-Safari
2. לחץ על כפתור השיתוף (📤)
3. גלול למטה → "הוסף למסך הבית"
4. לחץ "הוסף"

### 💻 מחשב (Chrome/Edge)
1. חפש אייקון התקנה (➕) בשורת הכתובת
2. לחץ "התקן"

## מה זה נותן לך?
✅ אייקון TurjiTrade במסך הבית  
✅ פתיחה מהירה כמו אפליקציה נייטיבית  
✅ מסך מלא בלי שורת כתובת  
✅ זמני טעינה מהירים יותר  
✅ עבודה חלקית במצב אופליין  

---

**קבצי PWA חשובים:**
- `/index.html` - דף HTML עם מטא-טאגים
- `/public/manifest.json` - הגדרות PWA
- `/public/sw.js` - Service Worker
- `/public/logo.svg` + icons - לוגו ואייקונים
- `/src/main.tsx` - רישום Service Worker
- `/src/app/components/InstallPrompt.tsx` - הודעת התקנה
